jChat+ is a Twitch and Kick chat overlay for OBS, XSplit, Meld, Streamlabs Desktop and any other streaming software that supports browser sources. It supports BetterTTV, FrankerFaceZ and 7TV emotes, Twitch badges, Kick badges, Kick emotes, smooth message animation, message fading, custom fonts, shadows, strokes and more.

Use this setup page to generate a browser source URL. Enable Kick if you want Twitch and Kick chat in the same overlay.
